Shift-click multi-select

Transient · familiar from Figma, Sketch, Adobe XD
How selection works
⇧ Click or ⌘ Click to toggle an element in/out of the selection. Click one without Shift to replace. Click empty stage to clear.
How editing works
StylePanel switches to multi-edit. Shared values render normally; diverging values leave the field blank. Typing a value applies to every selected element.
What it changes in code
Reuses modifyStyles() — runs once per selected testID inside a single debounced save. No schema change.
Try it
Shift-click the number keys below, then type a new Border Radius or Height — every selected key re-renders live. When the selected keys have different values, that field stays blank.
